DEMMo - Designer of Epidemic Math Models.
DEMMo – Конструктор Эпидемиологических Математических Моделей.

To run the program through python on any system, install the following libraries: 
Для запуска программы через интерпретатор python на любой системе необходимо установить следующие библиотеки:
PyQt5
Matplotlib
SciPy
Numpy

DEMMo is a program for create compartmental epidemic models such as SIR, SEIR and other. 
Full russian documentation in help/DEMMo_help.pdf
DEMMo – это программа для создания компартментальных эпидемиологических моделей, таких как SIR и SEIR.
Полная русская документация в help/DEMMo_help.pdf
